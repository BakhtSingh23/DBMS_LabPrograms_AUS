DBMS Lab Programs

Overview
This repository contains my Database Management Systems (DBMS) lab programs, developed as part of coursework at Aditya University.  
The programs are designed to strengthen understanding of fundamental and advanced DBMS concepts, with a focus on practical SQL implementation.

Key areas covered include:
- Table creation and schema design  
- Data manipulation using SQL (DDL, DML, DCL, TCL)  
- Joins, subqueries, and set operations  
- Views, indexing, and normalization  
- Stored procedures, functions, and triggers  

Lab Contents
- /schemas/ → Scripts for creating tables and inserting sample data  
- /queries/ → SQL queries for lab exercises and practice problems  
- /procedures/ → Stored procedures, functions, and triggers  
- /docs/ → ER diagrams, relational schema, and supporting documentation  

Purpose
These programs serve as a reference and practice resource for DBMS lab work, helping students and learners:
- Understand relational database concepts  
- Practice SQL queries with real examples  
- Prepare for lab exams and project work
